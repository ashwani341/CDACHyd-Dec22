What is Ajax?

Ajax is a capability of transfering info from a Web server as partial Requests instead of Full Request. A Partial Request is one that sends the request in the form of XML HTTP Request(XHR), here only partial part of the page is sent for the server and will update only that part of the page instead of the whole page sent for processing. IRCTC, GOOGLE MAPs, MS OFFIC 365 use this mechanism of updating partial sections of the Page instead of the whole page sent for the processing.
Ajax stands for Asynchronous JS and XML.
We would use Ajax to make calls to the server and fetch the data from the server. The data would be text based like JSON, XML or plain text.
In our current example, we are extracting text data from a Normal Text files.
To make this work, U should have a Web server, VS code has a Plugin to create LIVE Server on which U could test UR Application code.
