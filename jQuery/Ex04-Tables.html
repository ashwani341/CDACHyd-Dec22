$.each is a API of jQuery used to iterate thru the collection. Now thanks to ES6, we have different ways of iterating collections in JS. Earlier, we depended on the jQuery's $.each function to perform simple iterations on the collection objects.

How to debug JS code in a Browser

  • Open the Page in the browser and press F12(Developer Tools)
  • Move to the Source Tab and reload the page so that the HTML is loaded for debugging
  • Set the breakpoints to test the section of code that U want to monitor
  • Once the Action starts, the browser hits the break point and the App pauses the execution
  • Press F10 to execute each line of the code
  • If the Line contains a function call and U want to go into the function, press F11(Step Into)
  • Hover the mouse on the variables to evaluate their values
